AUTOMATIC TRANSMISSION REMOVAL/INSTALLATION [SJ6A-EL]

id051311249200

{: #wp1058566}

  1. Remove the battery cover.

  2. Disconnect the negative battery cable. (See BATTERY REMOVAL/INSTALLATION [L8, LF].)

  3. Drain the ATF. (See AUTOMATIC TRANSMISSION FLUID (ATF) REPLACEMENT [SJ6A-EL].)

  4. Loosen the starter installation bolts only enough that the starter is loose, but not removed. (See STARTER REMOVAL/INSTALLATION [L8, LF].)

  5. Remove in the order indicated in the table.

  6. Install in the reverse order of removal.

  7. Add ATF and, with the engine idling, inspect the ATF level and inspect for leakage. (See AUTOMATIC TRANSMISSION FLUID (ATF) LEVEL ADJUSTMENT [SJ6A-EL].)

  8. Inspect selector lever operation. (See SELECTOR LEVER INSPECTION.)

  9. Inspect for leakage of ATF from all connecting points.

  10. Perform the mechanical system test. (See MECHANICAL SYSTEM TEST [SJ6A-EL].)
