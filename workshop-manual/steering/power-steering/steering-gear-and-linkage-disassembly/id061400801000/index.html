Tie Rod Disassembly Note

  1. Unclamp the washer.

  2. Remove the tie rod.{: #wp1060633}

Pinion Plug, Pinion Shaft Component Disassembly Note

  1. Remove the clip and the dust cover, then remove the pinion plug using the SST.{: #wp1060670}

  1. Remove the pinion shaft component from the gear housing.{: #wp1060716}

Holder Disassembly Note

  1. Remove the crimp from the holder and cylinder ends using a hacksaw.{: #wp1060753}

  1. Remove the holder using the SST.{: #wp1060780}

Steering Rack, Oil Seal (Holder Side) Disassembly Note

  1. Install the SST to the gear housing.

  2. Remove the steering rack and oil seal (holder side) using a press.{: #wp1060845}

  1. Remove the oil seal (holder side) from the steering rack.

Oil Seal (Gear Housing Side), Inner Guide Disassembly Note

  1. Set the SSTs as shown.

  2. Install the SST to the gear housing.

  3. Remove the oil seal (gear housing side) and inner guide using a press.{: #wp1060930}
