REAR SHOCK ABSORBER AND COIL SPRING REMOVAL/INSTALLATION

id021400802600

Caution•  Performing the following procedures without first removing the ABS wheel-speed sensor may possibly cause an open circuit in the wiring harness if it is pulled by mistake. Before operations, remove the ABS wheel-speed sensor (axle side), and move the sensor away from the harnesses.

1.  Remove the trunk end trim. (See TRUNK END TRIM REMOVAL/INSTALLATION.)

2.  Remove the trunk mat. (See TRUNK MAT REMOVAL/INSTALLATION.)

3.  Remove the partition board. (See PARTITION BOARD REMOVAL/INSTALLATION.)

4.  Remove the trunk side trim. (See TRUNK SIDE TRIM REMOVAL/INSTALLATION.)

5.  When working on the left side of the vehicle, remove the fuel-filler pipe protector. (See FUEL TANK REMOVAL/INSTALLATION [L8, LF].)

6.  Remove in the order indicated in the table.

7.  Install in the reverse order of removal.

Caliper Removal Note

1.  Remove the caliper, and hang the cable out of the way.

Piston Rod Nut Removal Note

Warning•  Before removing the piston rod nut, secure the shock absorber and spring in the SSTs. Otherwise, the shock absorber and spring could fly off under tremendous pressure and cause serious injury or death, or damage to the vehicle.

1.  Set the SSTs using cloth to prevent the spring from being damaged.

amxzzw00003990

2.  Using the SSTs, compress the coil spring and remove the piston rod nut.

Coil Spring Installation Note

1.  Set the coil spring to the SSTs using cloth.

2.  Using the SSTs, compress the coil spring.

amxzzw00003991

3.  Install the shock absorber so that the lower end of the coil spring is seated on the step of the lower spring seat.
