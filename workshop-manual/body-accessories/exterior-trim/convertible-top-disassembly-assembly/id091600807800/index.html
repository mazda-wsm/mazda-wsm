Retainer Disassembly Note

  1. Mark around the retainer installation screws with paint before removing them.{: #wp1060491}

Top Fabric Disassembly Note

  1. Remove the top fabric from the front bow retainer.{: #wp1060528}

  1. Pull out the top fabric from the front header.{: #wp1060555}

  1. Remove the covers.{: #wp1060582}

  1. Remove the cable installation rivet using a drill.{: #wp1060609}

  1. Remove the cables from the cable guide.

  2. Remove the band installation rivet using a drill.{: #wp1060656}

  1. Remove the nut, then remove the cable end bracket.{: #wp1060683}

Rain Rail Disassembly Note

  1. Remove the rivets from the top fabric using a drill.

Spring A Disassembly Note

  1. Remove the bolt.{: #wp1060740}

  1. Remove the spacers, spring and collars from the link.

Spring B Disassembly Note

  1. Remove the bolt.{: #wp1060787}

  1. Pull the link in the direction of the arrow to remove the spacer and spring.

Rain Rail Assembly Note

  1. Degrease the rain rail using white gasoline.

  2. Install the insulation tape to the rivet installation hole of the rain rail.

  3. Secure the top fabric and rain rail with the rivet.{: #wp1060854}

  1. Flatten the stem using hammer.

Top Fabric Assembly Note

  1. Place the link onto the top fabric.

  2. Align the link with the set plate installation hole of the top fabric, and install the top fabric to the front header.{: #wp1060911}

  1. Thread the cable into the cable guide.{: #wp1060938}

  1. Set the aluminum rivet to the riveter, and then secure the cable to the link with the rivet.

  2. Install the covers.{: #wp1060975}

  1. Install the top fabric to the front bow.{: #wp1061002}

  1. Secure the top fabric to the front bow retainer using a rubber hammer.{: #wp1061029}

  1. Set the aluminum rivet to the riveter, and then secure the band to the link with the rivet.{: #wp1061056}

  1. Install the cable end bracket.{: #wp1061083}

Retainer Assembly Note

  1. Install the retainers to the link, aligning the retainer marks with the retainer installation screws.{: #wp1061120}
