DTC P2187 }{: #wp1061641

Fuel system too lean at idle {: #wp1058626}

DETECTION CONDITION {: #wp1058566}

• PCM monitors short term fuel trim (SHRTFT) and long term fuel trim (LONGFT) during closed loop fuel control at idle. If the LONGFT and the sum total of these fuel trims exceed preprogrammed criteria. PCM determines that fuel system is too lean at idle. {: #wp1058568} Diagnostic support note} • This is a continuous monitor. (Fuel system) {: #wp1058699} • The MIL illuminates if the PCM detects the above malfunctioning condition in two consecutive drive cycles or in one drive cycle while the DTC for the same malfunction has been stored in the PCM. {: #wp1058709} • PENDING CODE is available if the PCM detects the above malfunction conditions during the first drive cycle. {: #wp1058719} • FREEZE FRAME DATA is available. {: #wp1058729} • The DTC is stored in the PCM memory. {: #wp1058739

POSSIBLE CAUSE {: #wp1058641}

• Misfire {: #wp1058644} • A/F sensor deterioration {: #wp1058771} • A/F sensor heater malfunction {: #wp1058783} • MAF sensor malfunction {: #wp1058795} • Pressure regulator (built-in fuel injection pump) malfunction {: #wp1058805} • Fuel pump malfunction {: #wp1058816} • Fuel filter clogged or restricted {: #wp1058826} • Fuel leakage on fuel line from fuel delivery pipe and fuel pump {: #wp1058836} • Leakage exhaust system {: #wp1058846} • Purge solenoid valve malfunction {: #wp1058856} • Purge solenoid hoses improper connection {: #wp1058866} • Air suction in intake air system {: #wp1058876} • Insufficient engine compression {: #wp1058886} • Variable valve timing control system improper operation {: #wp1058896} • PCM malfunction {: #wp1058906}
