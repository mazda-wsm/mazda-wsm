DTC P2188 [L8, LF]

id0102b1812600

DTC P2188 }{: #wp1061281

Fuel system too rich at idle {: #wp1058626}

DETECTION CONDITION {: #wp1058566}

• PCM monitors short term fuel trim (SHRTFT), long term fuel trim (LONGFT) during closed loop fuel control at idle. If the LONGFT and the sum total of these fuel trims exceed preprogrammed criteria. PCM determines that fuel system is too rich at idle. {: #wp1058568} Diagnostic support note} • This is a continuous monitor. (Fuel system) {: #wp1058699} • The MIL illuminates if the PCM detects the above malfunctioning condition in two consecutive drive cycles or in one drive cycle while the DTC for the same malfunction has been stored in the PCM. {: #wp1058709} • PENDING CODE is available if the PCM detects the above malfunction conditions during the first drive cycle. {: #wp1058719} • FREEZE FRAME DATA is available. {: #wp1058729} • The DTC is stored in the PCM memory. {: #wp1058739

POSSIBLE CAUSE {: #wp1058641}

• Misfire {: #wp1058644} • A/F sensor deterioration {: #wp1058771} • A/F sensor heater malfunction {: #wp1058783} • MAF sensor malfunction {: #wp1058795} • Pressure regulator (built-in fuel injection pump) malfunction {: #wp1058805} • Fuel pump malfunction {: #wp1058816} • EGR valve stuck open {: #wp1058826} • Purge solenoid valve improper operation {: #wp1058836} • Purge solenoid valve malfunction (stuck open) {: #wp1058846} • Purge solenoid hoses improper connection {: #wp1058856} • PCV valve malfunction {: #wp1058866} • Variable valve timing control system improper operation {: #wp1058876} • PCM malfunction {: #wp1058886}
