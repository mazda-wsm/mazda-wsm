DTC P0011 [L8, LF]

id0102b1801600

{: #wp1059779}

DTC P0011 }{: #wp1061406

CMP timing over-advanced {: #wp1059842}

DETECTION CONDITION {: #wp1059782}

• The actual valve timing is over-advanced by 15 ° from the target valve timing for specified period when the oil control valve (OCV) is controlled in the maximum valve timing retard condition. {: #wp1059784} Monitoring condition - Engine coolant temperature is above 63 °C {145.4 °F} {: #wp1059927} Diagnostic support note} • This is a continuous monitor (CCM). {: #wp1059964} • The MIL illuminates if the PCM detects the above malfunction condition during the first drive cycle. {: #wp1059974} • PENDING CODE is available if the PCM detects the above malfunction condition. {: #wp1059984} • FREEZE FRAME DATA is available. {: #wp1059994} • The DTC is stored in the PCM memory. {: #wp1060004

POSSIBLE CAUSE {: #wp1059857}

• OCV malfunction {: #wp1059860} • Spool valve in OCV is stuck in advanced position {: #wp1060036} • Variable valve timing actuator is stuck in advanced position {: #wp1060046} • Loose timing chain or improper valve timing due to timing chain slippage {: #wp1060056} • PCM malfunction {: #wp1060066}
