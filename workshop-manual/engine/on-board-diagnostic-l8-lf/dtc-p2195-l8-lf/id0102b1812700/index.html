DTC P2195 [L8, LF]

id0102b1812700

{: #wp1058563}

DTC P2195 }{: #wp1061602

A/F sensor signal stuck lean {: #wp1058626}

DETECTION CONDITION {: #wp1058566}

• The PCM monitors the A/F sensor output current when the following conditions are met. If the average output current is more than 1.2 A for 25 s, the PCM determines that the A/F sensor signal remains lean. {: #wp1058703} MONITORING CONDITION - ECT: more than 70 °C {158 °F} {: #wp1058724}- Engine speed: 1,000-3,200 rpm {: #wp1058747}- MAF amount: 6-80 g/s {0.80-10.58 lb/min} {: #wp1058765}- Target A/F feedback system status: feedback control {: #wp1058792}- Output voltage from the HO2S: more than 0.2 V {: #wp1058804} Diagnostic support note} • This is an intermittent monitor (A/F sensor, HO2S). {: #wp1058830} • The MIL illuminates if the PCM detects the above malfunction condition in two consecutive drive cycles or in one drive cycle while the DTC for the same malfunction has been stored in the PCM. {: #wp1058842} • PENDING CODE is available if the PCM detects the above malfunction condition during the first drive cycle. {: #wp1058852} • FREEZE FRAME DATA is available. {: #wp1058862} • The DTC is stored in the PCM memory. {: #wp1058872

POSSIBLE CAUSE {: #wp1058641}

• A/F sensor malfunction {: #wp1058644} • Fuel injector malfunction {: #wp1058906} • Insufficient fuel line pressure {: #wp1058916} • Leakage exhaust gas {: #wp1058926} • Air suction at intake air system malfunction {: #wp1058936} • Leakage fuel {: #wp1058946} • MAF sensor malfunction {: #wp1058956} • ECT sensor malfunction {: #wp1058966} • PCM malfunction {: #wp1058976}
