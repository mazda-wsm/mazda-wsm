DTC P0133 [L8, LF]

id0102b1804800

{: #wp1058563}

DTC P0133 }{: #wp1061357

A/F sensor circuit problem {: #wp1058626}

DETECTION CONDITION {: #wp1058566}

• A/F sensor malfunction is detected by measuring of the average frequency of A/F sensor signal, time required to pass from rich to lean, time required to pass from lean to rich. If the average frequency of A/F sensor signal is more than the threshold value, the PCM determines that there is the A/F sensor circuit problem. {: #wp1058568} MONITORING CONDITIONS • Engine speed is } - A/F sensor heater, HO2S heater, A/F sensor, HO2S, and TWC Repair Verification Drive Mode {: #wp1058709}- Following conditions are met: {: #wp10587231,750-3,500 rpm. {: #wp1058733}• Charging efficiency is 19-63 % (at engine speed 2,500 rpm). {: #wp1058781}• Engine coolant temperature is above 70 °C {158 °F} {: #wp1058790}• A/F sensor heater monitor is completed. {: #wp1058813}• Fuel system loop status is closed loop fuel control. {: #wp1058825} Diagnostic support note} • This is an intermittent monitor. (A/F sensor, HO2S) {: #wp1058845} • The MIL illuminates if the PCM detects the above malfunction condition in two consecutive drive cycles or in one drive cycle while the DTC for the same malfunction has been stored in the PCM. {: #wp1058857} • DIAGNOSTIC MONITORING TEST RESULTS is available. {: #wp1058867} • PENDING CODE is available if the PCM detects the above malfunction condition during the first drive cycle. {: #wp1058877} • FREEZE FRAME DATA is available. {: #wp1058887} • The DTC is stored in the PCM memory. {: #wp1058897

POSSIBLE CAUSE {: #wp1058641}

• A/F sensor deterioration {: #wp1058644} • A/F sensor malfunction {: #wp1058931} • A/F sensor looseness {: #wp1058943} • Pressure regulator (built-in fuel pump unit) malfunction {: #wp1058955} • Fuel pump malfunction {: #wp1058966} • Clogged or restricted fuel filter (built-in fuel pump unit) {: #wp1058976} • Fuel leakage in fuel line between fuel distribution pipe and fuel pump {: #wp1058987} • Exhaust system leakage {: #wp1058997} • Purge solenoid valve malfunction {: #wp1059007} • Improper connection purge solenoid hose {: #wp1059017} • Insufficient compression {: #wp1059027} • Engine malfunction (Engine coolant leakage) {: #wp1059037}
