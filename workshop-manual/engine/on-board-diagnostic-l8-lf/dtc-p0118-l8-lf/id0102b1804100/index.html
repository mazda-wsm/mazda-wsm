DTC P0118 [L8, LF]

id0102b1804100

{: #wp1059779}

DTC P0118 }{: #wp1061562

ECT sensor circuit high input {: #wp1059842}

DETECTION CONDITION {: #wp1059782}

• The PCM monitors the ECT sensor signal at PCM terminal 2AH. If the PCM detects the ECT sensor voltage is above 4.58 V, the PCM determines that the ECT sensor circuit has malfunction. {: #wp1059784} Diagnostic support note} • This is a continuous monitor (Engine cooling system). {: #wp1059924} • The MIL illuminates if the PCM detects the above malfunction condition during first drive cycle. {: #wp1059934} • PENDING CODE is available if the PCM detects the above malfunction condition. {: #wp1059944} • FREEZE FRAME DATA is available. {: #wp1059954} • The DTC is stored in the PCM memory. {: #wp1059964

POSSIBLE CAUSE {: #wp1059857}

• ECT sensor malfunction {: #wp1059860} • Connect or terminal malfunction {: #wp1059996} • Open circuit in wiring harness between ECT sensor terminal A and PCM terminal 2AH {: #wp1060006} • Short to power supply in wiring harness between ECT sensor terminal A and PCM terminal 2AH {: #wp1060016} • Open circuit in wiring harness between ECT sensor terminal B and PCM terminal 2AY {: #wp1060026} • PCM malfunction {: #wp1060036}
