DTC P0703 [L8, LF]

id0102b1809900

{: #wp1059779}

DTC P0703 }{: #wp1062104

Brake switch input circuit problem {: #wp1059842}

DETECTION CONDITION {: #wp1059782}

• The PCM monitors changes in input voltage from the brake switch No.1. If the PCM does not the voltage changes while alternately accelerating and decelerating 8 times, the PCM determines that the brake switch No.1 circuit has a malfunction. {: #wp1059784} Diagnostic support note} • This is a continuous monitor (CCM). {: #wp1059924} • The MIL illuminates if the PCM detects the above malfunction condition in two consecutive drive cycles or in one drive cycle while the DTC for the same malfunction has been stored in the PCM. {: #wp1059934} • PENDING CODE is available if the PCM detects the above malfunction condition during the first drive cycle. {: #wp1059944} • FREEZE FRAME DATA is available. {: #wp1059954} • The DTC is stored in the PCM memory. {: #wp1059964

POSSIBLE CAUSE {: #wp1059857}

• Brake switch malfunction {: #wp1059860} • Poor connection of brake switch connector or PCM connector {: #wp1059996} • Short to power supply in wiring harness between brake switch No.1 terminal D and PCM connector terminal 1AB {: #wp1060006} • Open circuit in wiring harness between brake switch No.1 terminal D and PCM connector terminal 1AB {: #wp1060016} • Open circuit in wiring harness between battery positive terminal and brake switch No.1 terminal B {: #wp1060026} • PCM malfunction {: #wp1060036}
