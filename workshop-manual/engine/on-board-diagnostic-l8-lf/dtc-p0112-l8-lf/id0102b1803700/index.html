DTC P0112 [L8, LF]

id0102b1803700

{: #wp1059779}

DTC P0112 }{: #wp1061229

IAT sensor circuit low input {: #wp1059842}

DETECTION CONDITION {: #wp1059782}

• The PCM monitors the IAT sensor signal. If the PCM detects that the IAT sensor voltage is below 0.16 V, the PCM determines that the IAT sensor circuit has a malfunction. {: #wp1059784} Diagnostic support note} • This is a continuous monitor (CCM). {: #wp1059924} • The MIL illuminates if the PCM detects the above malfunction condition during first drive cycle. {: #wp1059934} • PENDING CODE is available if the PCM detects the above malfunction condition. {: #wp1059944} • FREEZE FRAME DATA is available. {: #wp1059954} • The DTC is stored in the PCM memory. {: #wp1059964

POSSIBLE CAUSE {: #wp1059857}

• IAT sensor malfunction {: #wp1059860} • Connector or terminal malfunction {: #wp1059996} • Short to ground in wiring harness between MAF/IAT sensor terminal D and PCM terminal 1AT {: #wp1060006} • IAT sensor signal circuit and IAT sensor ground circuit are shorted each other {: #wp1060018} • PCM malfunction {: #wp1060028}
