DTC P2177 [L8, LF]

id0102b1812300

{: #wp1058563}

DTC P2177 }{: #wp1061659

Fuel system too lean at off idle {: #wp1058626}

DETECTION CONDITION {: #wp1058566}

• PCM monitors short term fuel trim (SHRTFT), long term fuel trim (LONGFT) during closed loop fuel control at off-idle. If the LONGFT and the sum total of these fuel trims exceed preprogrammed criteria. PCM determines that fuel system is too lean at off-idle. {: #wp1058568} Diagnostic support note} • This is a continuous monitor. (Fuel system) {: #wp1058701} • The MIL illuminates if the PCM detects the above malfunctioning condition in two consecutive drive cycles or in one drive cycle while the DTC for the same malfunction has been stored in the PCM. {: #wp1058711} • PENDING CODE is available if the PCM detects the above malfunction conditions during the first drive cycle. {: #wp1058721} • FREEZE FRAME DATA is available. {: #wp1058731} • The DTC is stored in the PCM memory. {: #wp1058741

POSSIBLE CAUSE {: #wp1058641}

• Misfire {: #wp1058644} • A/F sensor deterioration {: #wp1058773} • A/F sensor heater malfunction {: #wp1058785} • MAF sensor malfunction {: #wp1058797} • Pressure regulator (built-in fuel injection pump) malfunction {: #wp1058807} • Fuel pump malfunction {: #wp1058818} • Fuel filter clogged or restricted {: #wp1058828} • Fuel leakage on fuel line from fuel delivery pipe and fuel pump {: #wp1058838} • Leakage exhaust system {: #wp1058848} • Purge solenoid valve improper operation {: #wp1058858} • Purge solenoid valve malfunction (stuck open) {: #wp1058868} • Purge solenoid hoses improper connection {: #wp1058878} • Air suction in intake air system {: #wp1058888} • Insufficient engine compression {: #wp1058898} • Variable valve timing control system improper operation {: #wp1058908} • PCM malfunction {: #wp1058918}
