MANIFOLD ABSOLUTE PRESSURE (MAP) SENSOR INSPECTION [L8, LF]

id0140i7800900

{: #wp1059779}

*Note*{: #wp1062432}

• Before performing the following inspection, make sure to follow the procedure as indicated in the troubleshooting flowchart. (See Troubleshooting Procedure.)

  1. Connect the M-MDS to the DLC-2.{: #wp1059805}

  1. Turn the ignition switch to the ON position (Engine off).

  2. Select MAP PID on the M-MDS.

  3. Verify that the MAP PID (pressure) and barometric pressure are practically equal.

• If not as verified, perform the "Circuit Open/Short Inspection".

  • If there is no open or short circuit, replace the MAP sensor. (See MANIFOLD ABSOLUTE PRESSURE (MAP) SENSOR REMOVAL/INSTALLATION [L8, LF].)

Circuit Open/Short Inspection

  1. Disconnect the PCM connector. (See PCM REMOVAL/INSTALLATION [L8, LF].)

  2. Inspect the following wiring harnesses for an open or short circuit. (Continuity Inspection)

Open circuit

• If there is no continuity, there is an open circuit. Repair or replace the wiring harness. {: #wp1060008}- MAP/boost air temperature sensor terminal A and PCM terminal 2AV {: #wp1060022}- MAP/boost air temperature sensor terminal C and PCM terminal 2AU {: #wp1060038}- MAP/boost air temperature sensor terminal D and PCM terminal 2AG

Short circuit

• If there is continuity, there is a short circuit. Repair or replace the wiring harness. {: #wp1060072}- MAP/boost air temperature sensor terminal A and power supply. {: #wp1060086}- MAP/boost air temperature sensor terminal C and body ground. {: #wp1060102}- MAP/boost air temperature sensor terminal D and power supply. {: #wp1060114}- MAP/boost air temperature sensor terminal D and body ground.
