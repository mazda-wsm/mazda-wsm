NO.19 EXHAUST SMOKE [L8, LF]

id0103a0803900

{: #wp1059779}

19 }{: #wp1061436

EXHAUST SMOKE {: #wp1059842}

DESCRIPTION {: #wp1059782}

Blue, black, or white smoke from exhaust system {: #wp1059784}

POSSIBLE CAUSE {: #wp1059857}

Blue smoke (Burning oil): {: #wp1059860} • PCV valve malfunction {: #wp1059915} • Engine internal oil leakage {: #wp1059929} White smoke (Water in combustion): {: #wp1059939} • Cooling system malfunction (coolant loss) {: #wp1059949} • Engine internal coolant leakage {: #wp1059963} Black smoke (Rich fuel mixture): {: #wp1059973} • Air cleaner restriction {: #wp1059983} • Intake-air system is collapsed or restricted. {: #wp1059997} • Fuel return line is restricted. {: #wp1060009} • Excessive fuel pressure {: #wp1060019} • Improper engine compression {: #wp1060029} • Injector fuel leakage {: #wp1060039} • Ignition system malfunction {: #wp1060049} Warning(See } The following troubleshooting flow chart contains fuel system diagnosis and repair procedures. Read following warnings before performing fuel system services: {: #wp1061358} • Fuel vapor is hazardous. It can easily ignite, causing serious injury and damage. Always keep sparks and flames away from fuel. {: #wp1060069}• Fuel line spills and leakage are dangerous. Fuel can ignite and cause serious injuries or death and damage. Fuel can also irritate skin and eyes. To prevent this, always complete "BEFORE SERVICE PRECAUTION" and "AFTER SERVICE PRECAUTION" described in this manual. {: #wp1060083BEFORE SERVICE PRECAUTION [L8, LF].) {: #wp1060101}(See AFTER SERVICE PRECAUTION [L8, LF].) {: #wp1060115} Caution} • Disconnecting/connecting quick release connector without cleaning it may possibly cause damage to fuel pipe and quick release connector. Always clean quick release connector joint area before disconnecting/connecting, and make sure that it is free of foreign material. {: #wp1061359

Diagnostic procedure

STEP {: #wp1060273}

INSPECTION {: #wp1060275}

RESULTS {: #wp1060277}

ACTION {: #wp1060279}

1 {: #wp1060167}

What color is smoke coming from the exhaust system? {: #wp1060169}

Blue {: #wp1060225}

Burning oil is indicated. {: #wp1060228} Go to the next step. {: #wp1060396}

White {: #wp1060308}

Water in combustion is indicated. {: #wp1060311} Go to Step 3. {: #wp1060412}

Black {: #wp1060321}

Rich fuel mixture is indicated. {: #wp1060324} Go to Step 4. {: #wp1060428}

2 {: #wp1060328}

Remove and shake the PCV valve. {: #wp1060331} Does the PCV valve rattle? {: #wp1060444}

Yes {: #wp1060334}

Inspect for the following: {: #wp1060337} • Damaged valve guide, stems or valve seals {: #wp1060460} • Blocked oil drain passage in cylinder head {: #wp1060474} • Piston ring is not seated, seized or worn. {: #wp1060484} • Damaged cylinder bore {: #wp1060494} If other driveability symptoms are present, return to diagnostic index to service any additional symptoms. {: #wp1060504}

No {: #wp1060347}

Replace the PCV valve. {: #wp1060350}

3 {: #wp1060518}

Does the cooling system hold pressure? {: #wp1060521}

Yes {: #wp1060524}

Inspect for the following: {: #wp1060527} • Cylinder head gasket leakage {: #wp1060566} • Intake manifold gasket leakage {: #wp1060580} • Cracked or porous engine block {: #wp1060590} If other driveability symptoms are present, return to diagnostic index to service any additional symptoms. {: #wp1060600}

No {: #wp1060537}

Inspect for cause. {: #wp1060540}

4 {: #wp1060614}

Inspect for the following: {: #wp1060617} • Air cleaner restriction {: #wp1060652} • Collapsed or restricted intake-air system {: #wp1060666} • Restricted fuel return line {: #wp1060678} Are all items normal? {: #wp1060688}

Yes {: #wp1060620}

Go to the next step. {: #wp1060623}

No {: #wp1060633}

Service if necessary. {: #wp1060636} Repeat Step 4. {: #wp1060714}

5 {: #wp1060718}

Connect the M-MDS to the DLC-2. {: #wp1060721} Retrieve any continuous memory, KOEO and KOER DTCs using M-MDS. {: #wp1060758} Are there any DTCs displayed? {: #wp1060765}

Yes {: #wp1060724}

DTC is displayed: {: #wp1060727} Go to the appropriate DTC inspection. {: #wp1060785} (See DTC TABLE [L8, LF].) {: #wp1060791}

No {: #wp1060737}

No DTC is displayed: {: #wp1060740} Go to the next step. {: #wp1060819}

6 {: #wp1060823}

Install fuel pressure gauge between the fuel pipe and fuel distributor. {: #wp1060826} Start the engine and idle it. {: #wp1060869} Measure fuel line pressure during idle. {: #wp1060875} Is fuel line pressure correct during idle? {: #wp1060881} (See FUEL LINE PRESSURE INSPECTION [L8, LF].) {: #wp1060887}

Yes {: #wp1060829}

Go to the next step. {: #wp1060832}

No {: #wp1060844}

Low: {: #wp1060847} Inspect the fuel line for clogging. {: #wp1060925} • If there is no malfunction, replace the fuel pump unit. {: #wp1060931} (See FUEL PUMP UNIT REMOVAL/INSTALLATION [L8, LF].) {: #wp1060945} High: {: #wp1060959} Replace the fuel pump unit. {: #wp1060969} (See FUEL PUMP UNIT REMOVAL/INSTALLATION [L8, LF].) {: #wp1060975}

7 {: #wp1060987}

Perform the spark test. {: #wp1060990} (See Spark Test.) {: #wp1061057} Is strong blue spark visible at each cylinder? {: #wp1061071}

Yes {: #wp1060993}

Inspect the CMP sensor. {: #wp1060996} (See CAMSHAFT POSITION (CMP) SENSOR INSPECTION [L8, LF].) {: #wp1061087}

No {: #wp1061014}

Repair or replace the malfunctioning part according to spark test result. {: #wp1061017}

8 {: #wp1061109}

Verify test results. {: #wp1061112} • If normal, return to diagnostic index to service any additional symptoms. {: #wp1061158} (See ENGINE SYMPTOM TROUBLESHOOTING [L8, LF].) {: #wp1061172} • If malfunction remains, inspect related Service information perform repair or diagnosis. {: #wp1061186} - If vehicle repaired, troubleshooting completed. {: #wp1061196}- If vehicle not repaired or additional diagnostic information not available, replace the PCM. {: #wp1061210}(See PCM REMOVAL/INSTALLATION [L8, LF].) {: #wp1061220}
