DETECTION CONDITION {: #wp1059784}

• Vehicle speed is 30 km/h {18.6 mph} or more, and no range signal is input from the TR switch. {: #wp1059786} Diagnostic support note:} • The MIL illuminates if the TCM detects the above malfunction condition in two consecutive drive cycles. {: #wp1059932} • PENDING CODE is available. {: #wp1059942} • FREEZE FRAME DATA is available. {: #wp1059952} • The AT warning light illuminates. {: #wp1059962} • The DTC is stored in the TCM memory. {: #wp1059972
