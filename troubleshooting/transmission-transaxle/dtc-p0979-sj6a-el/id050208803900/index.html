DTC P0979 [SJ6A-EL]

id050208803900

{: #wp1059781}

DTC P0979 }{: #wp1061648

Shift solenoid C circuit malfunction (short to ground) {: #wp1059844}

DETECTION CONDITION {: #wp1059784}

• Short to ground in shift solenoid C signal system (while TCM monitors solenoid output voltage, the voltage that differs from the signal output by CPU in TCM is detected). {: #wp1059786} Diagnostic support note:} • The MIL illuminates if the TCM detects the above malfunction condition during first drive cycle. {: #wp1059917} • A PENDING CODE is not available. {: #wp1059927} • FREEZE FRAME DATA is available. {: #wp1059937} • The AT warning light illuminates. {: #wp1059947} • The DTC is stored in the TCM memory. {: #wp1059957

POSSIBLE CAUSE {: #wp1059859}

• Short to ground in wiring harness between shift solenoid C terminal A and TCM terminal 1AA {: #wp1059862} • Shift solenoid C malfunction {: #wp1059989} • Damaged connector between shift solenoid C and TCM {: #wp1059999} • TCM malfunction {: #wp1060009}
