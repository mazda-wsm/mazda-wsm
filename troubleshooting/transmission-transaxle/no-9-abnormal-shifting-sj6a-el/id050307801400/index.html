NO.9 ABNORMAL SHIFTING [SJ6A-EL]

id050307801400

{: #wp1059780}

9 }{: #wp1060973

Abnormal shifting {: #wp1059785}

DESCRIPTION {: #wp1059858}

• Shift incorrectly (incorrect shift pattern). {: #wp1059861}

POSSIBLE CAUSE {: #wp1059865}

• There is a malfunction in the signal circuit which controls shifting (Throttle position signal, turbine sensor, VSS), the control valve is stuck, or the clutch circuit is stuck. {: #wp1059868} - Clutch slippage, burnt (C1 clutch, C2 clutch, C3 clutch, B1 brake, B2 brake, B3 brake) {: #wp1059936} • Line pressure low {: #wp1059950}• Incorrect throttle position signal {: #wp1059964}• Incorrect accelerator pedal position signal {: #wp1059974}• Incorrect engine speed signal {: #wp1059984}• Incorrect engine coolant temperature signal {: #wp1059994}• Incorrect engine torque signal {: #wp1060004}• VSS malfunction {: #wp1060014}• Turbine sensor malfunction {: #wp1060024}• Sensor GND malfunction {: #wp1060034}• TFT sensor malfunction {: #wp1060044}• Line pressure control solenoid malfunction {: #wp1060054}• Body GND malfunction {: #wp1060064}• Control valve body malfunction {: #wp1060074}• TCM malfunction {: #wp1060084} *Note*{: #wp1060094} • Before following the troubleshooting steps, make sure that the Automatic Transmission On-Board Diagnostic and Automatic Transmission Basic Inspection are conducted. (See BASIC INSPECTION [SJ6A-EL].) {: #wp1060896}

Diagnostic procedure

STEP {: #wp1060239}

INSPECTION {: #wp1060241}

ACTION {: #wp1060245}

1 {: #wp1060133}

• Inspect for continuity between the TCM terminal 1P, 1X and battery negative terminal. {: #wp1060135} • Is there continuity? {: #wp1060355}

Yes {: #wp1060191}

Go to the next step. {: #wp1060194}

No {: #wp1060274}

Repair or replace ground circuit. {: #wp1060277}

2 {: #wp1060281}

• Access the following PCM PIDs using the M-MDS. {: #wp1060284} - RPM {: #wp1060404}- APP {: #wp1060418}- TP {: #wp1060428}- ECT {: #wp1060438}- ENG_TRQ {: #wp1060448} • Are PIDs value normal? {: #wp1060460}

Yes {: #wp1060287}

Go to the next step. {: #wp1060290}

No {: #wp1060300}

Inspect and repair suspected PID related part. {: #wp1060303}

3 {: #wp1060307}

• Inspect the TFT sensor. {: #wp1060310} (See TRANSMISSION FLUID TEMPERATURE (TFT) SENSOR INSPECTION [SJ6A-EL].) {: #wp1060521} • Is the TFT sensor normal? {: #wp1060535}

Yes {: #wp1060313}

Go to the next step. {: #wp1060316}

No {: #wp1060494}

Replace the TFT sensor. {: #wp1060497}

4 {: #wp1060563}

• Inspect the line pressure control solenoid. {: #wp1060566} (See SOLENOID VALVE INSPECTION [SJ6A-EL].) {: #wp1060617} • Is the line pressure control solenoid normal? {: #wp1060631}

Yes {: #wp1060569}

Inspect the TCM terminal for bend, damage, corrosion or poor contact. {: #wp1060572}

No {: #wp1060584}

Inspect the coupler component. {: #wp1060587} If normal, replace the control valve body. {: #wp1060661} (See CONTROL VALVE BODY REMOVAL [SJ6A-EL].) {: #wp1060667} (See CONTROL VALVE BODY INSTALLATION [SJ6A-EL].) {: #wp1060681}

5 {: #wp1060693}

• Verify test results. {: #wp1060696} - If normal, return to the diagnostic index to service any additional symptoms. {: #wp1060742}- If malfunction remains, inspect the related Service Information and perform repair or diagnosis. {: #wp1060756} • If the vehicle is repaired, troubleshooting completed. {: #wp1060766}• If the vehicle is not repaired or additional diagnostic information is not available, replace the TCM. {: #wp1060780}
