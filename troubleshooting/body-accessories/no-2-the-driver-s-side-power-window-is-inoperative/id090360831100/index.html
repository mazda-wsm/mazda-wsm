NO. 2 THE DRIVER'S SIDE POWER WINDOW IS INOPERATIVE

id090360831100

{: #wp1059778}

2 }{: #wp1061319

The driver's side power window is inoperative. {: #wp1059783}

POSSIBLE CAUSE {: #wp1059856}

• Power supply circuit or ground circuit malfunction {: #wp1059859} - Burnt fuse (B+) {: #wp1059914}- Open or short circuit in wiring harness between fuse (B+) and power window main switch {: #wp1059930}- Open or short circuit in wiring harness between power window main switch and power window motor {: #wp1059942}- Open or short circuit in wiring harness between power window main switch and ground {: #wp1059952} • Power window main switch malfunction {: #wp1059962} • Power window motor malfunction {: #wp1059972} • Power window regulator malfunction {: #wp1059982}

Diagnostic procedure

STEP {: #wp1060106}

INSPECTION {: #wp1060108}

ACTION {: #wp1060112}

1 {: #wp1060000}

• Turn the ignition switch to the ON position. {: #wp1060002} • Does the LED on the power window main switch illuminate? {: #wp1060222}

Yes {: #wp1060058}

Go to Step 5. {: #wp1060061}

No {: #wp1060141}

Go to the next step. {: #wp1060144}

2 {: #wp1060148}

• Operate passenger's side power window using the power window main switch. {: #wp1060151} • Does the power window operate properly? {: #wp1060272}

Yes {: #wp1060154}

Go to the next step. {: #wp1060157}

No {: #wp1060167}

Inspect for an open or short circuit in the following wiring harnesses. Inspect the connector connections (damage/pulled-out pins, corrosion): {: #wp1060170} • P.WIND 20 A fuse-Power window main switch terminal B {: #wp1060305} • Power window main switch terminal G-ground. {: #wp1060321} Repair or replace if necessary. {: #wp1060333}

3 {: #wp1060174}

• Is the P.WIND 20 A fuse normal? {: #wp1060177}

Yes {: #wp1060180}

Replace the power window main switch. {: #wp1060183} (See POWER WINDOW MAIN SWITCH INSPECTION [EXCEPT POWER RETRACTABLE HARDTOP].) {: #wp1060380} (See POWER WINDOW MAIN SWITCH INSPECTION [POWER RETRACTABLE HARDTOP].) {: #wp1060394}

No {: #wp1060343}

Inspect the following: {: #wp1060346} • Short circuit in B+ power supply wiring harness {: #wp1060418} • Short circuit in power window motor {: #wp1060434} Repair or replace if necessary. {: #wp1060444} Replace with the appropriate standard fuse. {: #wp1060450} Then, go to the next step. {: #wp1060456}

4 {: #wp1060460}

• Initialize the power window system. {: #wp1060463} • Operate the power window system. {: #wp1060522} • Do the power windows operate properly? {: #wp1060532}

Yes {: #wp1060466}

Troubleshooting is completed. {: #wp1060469}

No {: #wp1060483}

Re-confirm the symptom and go to Step 1. {: #wp1060486}

5 {: #wp1060561}

• Measure the voltage at the power window main switch. (power window motor output terminal) while operating the power window using the power window main switch. {: #wp1060564} • Is the voltage B+? {: #wp1060623} L.H.D.model: } open: terminal J/close: terminal I {: #wp1060645R.H.D.model:} open: terminal F/close: terminal D {: #wp1060667

Yes {: #wp1060567}

Go to the next step. {: #wp1060570}

No {: #wp1060584}

Replace the power window main switch. {: #wp1060587} (See POWER WINDOW MAIN SWITCH INSPECTION [EXCEPT POWER RETRACTABLE HARDTOP].) {: #wp1060699} (See POWER WINDOW MAIN SWITCH INSPECTION [POWER RETRACTABLE HARDTOP].) {: #wp1060713}

6 {: #wp1060725}

• Measure the voltage at the power window motor. (battery power supply terminal) while operating the power window using the power window main switch. {: #wp1060728} • Is the voltage B+? {: #wp1060803} (Open: terminal A/ close: terminal G) {: #wp1060815}

Yes {: #wp1060731}

Go to the next step. {: #wp1060734}

No {: #wp1060752}

Inspect for an open or short circuit in the wiring harness between the power window main switch and the power window motor. {: #wp1060755} Inspect the power window main switch and power window motor connector connections. (damage/pulled-out pins, corrosion) {: #wp1060843} Repair or replace if necessary. {: #wp1060852}

7 {: #wp1060856}

• Operate the driver's side power window using the power window main switch. {: #wp1060859} • Does the power window motor operate (rotate)? {: #wp1060936} Caution} • If the power window motor temperature is high, the motor may not rotate due to the motor internal bimetal function. Leave it untouched for about 3 min to cool it down, then reinspect. {: #wp1061242

Yes {: #wp1060862}

Go to the next step. {: #wp1060865}

No {: #wp1060883}

Replace the power window motor. (See POWER WINDOW REGULATOR REMOVAL/INSTALLATION.) {: #wp1060886}

8 {: #wp1060990}

• Remove the door glass from the carrier plate. {: #wp1060993} • Make sure that the door glass moves smoothly using your hand. {: #wp1061076} • Does the door glass move smoothly? {: #wp1061086}

Yes {: #wp1060996}

Replace the power window regulator. (See POWER WINDOW REGULATOR REMOVAL/INSTALLATION.) {: #wp1060999}

No {: #wp1061019}

Inspect for a bent regulator guide or other possible malfunction. If normal, replace the door glass run-channel. {: #wp1061022}
