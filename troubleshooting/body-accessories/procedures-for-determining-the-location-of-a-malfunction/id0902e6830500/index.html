Repair Procedure

  1. Inspect the connector of malfunctioning module.

• If there is any malfunction, repair or replace the connector.

  1. Inspect the malfunctioning wiring harnesses as follow:

• If there is any malfunction, repair or replace the wiring harnesses. {: #wp1066930}• If there is no malfunction, replace the malfunctioning module.

  • Short to GND {: #wp1066954}- Short to power supply {: #wp1066968}- Twisted pair short each other {: #wp1066978}- Open circuit

  1. Make sure to reconnect all disconnected connectors.

  2. Clear the CAN system related DTCs using the M-MDS.

  3. Verify if the CAN system related DTCs are displayed using the M-MDS.

• If the same following DTCs are present, replace the malfunctioning module.

  • U0073 (PCM, TCM, DSC HU/CM, ABS HU/CM, keyless control module, instrument cluster, power retractable hardtop control module) {: #wp1067044}- U2516 (steering angle sensor)

• If other DTC is present, perform the appropriate DTC inspection. (See DTC TABLE [MULTIPLEX COMMUNICATION SYSTEM].)
