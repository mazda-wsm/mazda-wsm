NO. 4 ALL POWER WINDOWS ARE INOPERATIVE

id090360870200

{: #wp1059776}

4 }{: #wp1061132

All power windows are inoperative. {: #wp1059781}

POSSIBLE CAUSE {: #wp1059854}

• Power supply circuit or ground circuit malfunction {: #wp1059857} - Burnt fuse (B+) {: #wp1059910}- Open or short circuit in wiring harness between ignition switch (IG1) and power window main switch {: #wp1059926}- Open or short circuit in wiring harness between power window main switch and power window motor {: #wp1059936}- Open or short circuit in wiring harness between power window main switch and ground {: #wp1059946} • Power window main switch malfunction (switch malfunction) {: #wp1059956} • Power window motor malfunction {: #wp1059966}

Diagnostic procedure

STEP {: #wp1060090}

INSPECTION {: #wp1060092}

ACTION {: #wp1060096}

1 {: #wp1059984}

• Operate all power windows other than driver's side window using the power window main switch. {: #wp1059986} • Does any power window operate? {: #wp1060208}

Yes {: #wp1060042}

Go to Step 4. {: #wp1060045}

No {: #wp1060125}

Go to the next step. {: #wp1060128}

2 {: #wp1060132}

• Operate the driver's side power window using the power window main switch. {: #wp1060135} • Does the power window operate? {: #wp1060258}

Yes {: #wp1060138}

Go to the next step. {: #wp1060141}

No {: #wp1060151}

Inspect for an open circuit in the wiring harness between the power window main switch and the body ground. {: #wp1060154} Inspect the power window main switch connector connection. (damage/pulled-out pins, corrosion) {: #wp1060288} Repair or replace if necessary {: #wp1060297}

3 {: #wp1060158}

• Is the P.WIND 20 A fuse normal? {: #wp1060161}

Yes {: #wp1060164}

Replace with the appropriate standard fuse. {: #wp1060167} If the fuse is melted, inspect the wiring harness for a short to ground. Repair or replace the wiring harness, then replace the fuse. {: #wp1060344}

No {: #wp1060307}

Replace the power window main switch. {: #wp1060310} (See POWER WINDOW MAIN SWITCH INSPECTION [EXCEPT POWER RETRACTABLE HARDTOP].) {: #wp1060360} (See POWER WINDOW MAIN SWITCH INSPECTION [POWER RETRACTABLE HARDTOP].) {: #wp1060374}

4 {: #wp1060386}

• Operate the power window using the power window subswitch {: #wp1060389} • Does the power window motor operate (rotate)? {: #wp1060448} Caution} • If the power window motor temperature is high, the motor may not rotate due to the motor internal bimetal function. Leave it untouched for about 3 min to cool it down, then reinspect. {: #wp1061052

Yes {: #wp1060392}

Go to the next step. {: #wp1060395}

No {: #wp1060409}

Replace the power window motor. (See POWER WINDOW REGULATOR REMOVAL/INSTALLATION.) {: #wp1060412}

5 {: #wp1060502}

• Remove the door glass from the carrier plate. {: #wp1060505} • Make sure that the door glass moves smoothly using your hand. {: #wp1060572} • Does the door glass move smoothly? {: #wp1060582}

Yes {: #wp1060508}

Replace the power window regulator guide. {: #wp1060511}

No {: #wp1060527}

Inspect for a bent regulator guide or other possible malfunction. If normal, replace the glass run channel {: #wp1060530}

6 {: #wp1060610}

*Note*{: #wp1060613} • Do not operate the power window main switch during the following inspection. {: #wp1061053} • Inspect the continuity between power window main switch terminal (up-side I, D) and ground. {: #wp1060684} • Is there continuity? {: #wp1060699}

Yes {: #wp1060616}

Go to the next step. {: #wp1060619}

No {: #wp1060635}

Replace the power window main switch. {: #wp1060638} (See POWER WINDOW MAIN SWITCH INSPECTION [EXCEPT POWER RETRACTABLE HARDTOP].) {: #wp1060729} (See POWER WINDOW MAIN SWITCH INSPECTION [POWER RETRACTABLE HARDTOP].) {: #wp1060743}

7 {: #wp1060755}

*Note*{: #wp1060758} • Do not operate the power window main switch during the following inspection. {: #wp1061054} • Inspect the continuity between power window main switch terminal (down-side F, J) and ground. {: #wp1060845} • Is there continuity? {: #wp1060860}

Yes {: #wp1060761}

Inspect for an open or short circuit in the wiring harness between the power window main switch and power window subswitch. {: #wp1060764} Inspect the power window main switch and subswitch connector connections. (damage/pulled-out pins, corrosion) Repair or replace if necessary. {: #wp1060880}

No {: #wp1060784}

Replace the power window main switch. {: #wp1060787} (See POWER WINDOW MAIN SWITCH INSPECTION [EXCEPT POWER RETRACTABLE HARDTOP].) {: #wp1060899} (See POWER WINDOW MAIN SWITCH INSPECTION [POWER RETRACTABLE HARDTOP].) {: #wp1060913}
