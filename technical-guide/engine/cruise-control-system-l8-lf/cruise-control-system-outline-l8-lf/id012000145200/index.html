CRUISE CONTROL SYSTEM OUTLINE [L8, LF]

id012000145200

{: #wp1059779}

• The cruise control system enables driving at a constant speed by setting the vehicle speed with the cruise control switch instead of operating the accelerator pedal.

• The PCM controls the throttle valve actuator to maintain the vehicle at a constant speed.

Component and function

Component }{: #wp1059813

Function {: #wp1059950}

Installation location {: #wp1059954}

ABS DSC HU/CM (CAN communication: Vehicle speed signal) {: #wp1059816}

The vehicle speed signal is sent to the PCM from either the ABS HU/CM or DSC HU/CM. {: #wp1059874}

Engine compartment {: #wp1059880}

Cruise control {: #wp1059981} switch {: #wp1060120}

ON OFF {: #wp1059984}

ON OFF {: #wp1059987}

This is the main switch of the cruise control system. Turning the ON OFF switch to on switches the cruise control system to standby status. {: #wp1059990}

Steering wheel {: #wp1059993}

SET/- {: #wp1060000}

SET {: #wp1060003}

When the vehicle speed exceeds 27 km/h {17 mph} during normal driving (cruise control system is in standby status) and the SET/- switch is released after it is pressed, the PCM stores the vehicle speed at the time the switch is released and the cruise control begins. {: #wp1060006}
  • {: #wp1060019}

Tapping the SET/- switch (tap-down operation) or continuously pressing it during cruise control decreases the set vehicle speed. {: #wp1060022}

RES/+ {: #wp1060032}

RES {: #wp1060035}

If the RES/+ switch is pressed while the cruise control is in standby status (PCM has stored a set vehicle speed) and the vehicle speed exceeds 27 km/h {17 mph} during normal driving, the cruise control system activates to control the vehicle speed to the set vehicle speed. {: #wp1060038}
  • {: #wp1060051}

Tapping the RES/+ switch (tap-up operation) or continuously pressing it during cruise control increases the set vehicle speed. {: #wp1060054}

CANCEL {: #wp1060100}

CANCEL {: #wp1060103}

Pressing the CANCEL switch during cruise control switches the cruise control system to standby status (Set vehicle speed is saved). {: #wp1060106}

Brake switch {: #wp1060239}

Depressing the brake pedal during cruise control switches the cruise control system to standby status (Set vehicle speed is saved). {: #wp1060245}

Brake pedal {: #wp1060251}

CPP switch (MT) {: #wp1060270}

Depressing the clutch pedal during cruise control switches the cruise control system to standby status (Set vehicle speed is saved). {: #wp1060276}

Clutch pedal {: #wp1060282}

Neutral switch (MT) {: #wp1060301}

Shifting to neutral during cruise control switches the cruise control system to standby status (Set vehicle speed is saved). {: #wp1060307}

Manual transaxle {: #wp1060313}

PCM {: #wp1060332}

• The cruise control system activates or stops based on the cruise control switch ON OFF signal. {: #wp1060338} • The cruise control duty signal, which is based on each input signal, is sent to the throttle valve actuator. {: #wp1060368}

Engine compartment {: #wp1060344}

Throttle valve actuator {: #wp1060381}

The duty signal sent from the PCM adjusts the throttle valve opening angle. {: #wp1060387}

Throttle body {: #wp1060393}

Cruise main indicator light {: #wp1060412}

This illuminates an amber color while the cruise control system is on standby. {: #wp1060418}

Instrument cluster {: #wp1060424}

Cruise set indicator light {: #wp1060438}

This illuminates a green color while the cruise control system is in control status. {: #wp1060444}
