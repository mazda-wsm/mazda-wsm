WARNING/GUIDANCE FUNCTION OPERATION

id0914b5102800

{: #wp1058562}

• If the system is operated improperly, it warns the driver using the indicator light in the instrument cluster, buzzer sound, and keyless beeper in the trunk compartment.

• The operation condition of the advanced keyless system is indicated by the indicator light and beeper sound to guide user's operation.
